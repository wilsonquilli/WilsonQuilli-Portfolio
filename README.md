# Wilson Quilli Portfolio

A personal portfolio website built with React and Vite to showcase my projects, experience, skills, and contact information.

## Overview

This portfolio features:

- A custom hero section with dark/light theme switching
- Smooth scroll and section reveal animations
- A split-layout About section
- Projects, experience, and contact sections
- Resume access through a public PDF asset
- Responsive layouts designed for desktop and mobile

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Tailwind CSS
- React Icons

## Deployment

### Vercel

This project is ready for Vercel as a standard Vite app.

## Customization Notes

- Main page content lives in `src/App.jsx`
- Global base styles live in `src/index.css`
- Resume file is served from `public/Wilson Quilli Resume.pdf`
- Favicon is served from `public/Logo.png`
